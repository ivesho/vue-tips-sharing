## ☀️ MorningRoutineCompare.vue — Vue 同步 vs 非同步任務模擬器

這是一個用來教學「同步流程」與「非同步流程」差異的 Vue 小模擬器，功能如下：

### ✅ 功能特色
- 左右分欄展示「同步」與「非同步」執行結果
- 自動滾動到底，
- 執行結束後允許重新開始，清除前一次紀錄

### 📊 結果範例
```
☑️ 同步流程總共花費：39.8 分鐘
⚡ 非同步流程總共花費：25.8 分鐘
🏁 非同步效率提升了 14 分鐘（+38.18%）
```
